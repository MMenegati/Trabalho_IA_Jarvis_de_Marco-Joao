# Trabalho_IA_Jarvis_de_Marco-Joao
O projeto utiliza o modelo Gemma 12B para auxiliar usuários em tarefas acadêmicas como consulta de materiais de estudo, gerenciamento de agenda, organização de tarefas e planejamento de estudos personalizados. Tecnologias: Python, Gemma12B, FAISS, Sentence Transformers, SQLite e OpenAISDK
